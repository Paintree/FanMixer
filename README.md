# fan_controller_board
Cheap PWM fan controller based on Blue Pill

Supported MCUs: STM32F103C6T6

commands:

Build project:
$ make build

Clean build files:
$ make clean

Test project:
$ make test

Flash board:
$ make flash

Required dependencies:
arm-none-eabi-gcc
arm-none-eabi-g++
arm-none-eabi-objcopy"
arm-none-eabi-objdump"
make
cmake
openocd
gdb
